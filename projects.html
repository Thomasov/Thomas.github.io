Olympic Weights

Fall 2015

Olympic Weights is a mobile application written in Apache Cordova with an accompanying .NET web application for admin functionality. I also rolled my own generic MVC framework to be used in this and future Cordova apps. Once fully tested, I plan to open source the framework for others to use and modify as they will. Unfortunately, due the nature of the application, I'm not allowed to give out logins to anyone who wants to see it. But I've uploaded some screenshots of the mobile app to get an idea of the theme. All of the screenshots are from the iOS version of the app.

Auburn Office of Accessibility

July 2014

Auburn University's Office of Accessibility C# .NET web application that I wrote early in my career as a web developer. There's not a lot to it, but I'm proud of a lot of the little things in it. It boasts a dynamic system for adding pages that the client can do easily without ever touching any code. They can also manage many other aspects of the site, including but not limited to the number of links on the home page, images in the slideshow, and news articles on the home page. All of this information is stored in an SQL Server database.

Endless Runner

July 2015

This is an endless runner game I built in Unity 5 by following a tutorial by Jesse Freeman on Lynda.com and then expanding on it. All of the implementation was done in C#. The system was built modularly so that the various components, such as the time manager, object pool, and menu manager, could be reused in other future projects. The artwork is almost exclusively not mine (credit to Jesse Freeman), but I did draw the art for the barrel that sometimes appears as an obstacle and the artwork for the pause screen during a forray I had into pixel art.

Plight of the Tauren

Around 2008

A custom Warcraft III campaign based on the conflict between the Tauren and the Centaur in the Barrens. Created in the WCIII World Editor, it has scripted cut scenes and optional quests. Plays very similarly to the already in-game campaigns. All art is owned by Blizzard Entertainment, with the exception of the model of the main character, the Gnome Mage, who was created by user Norinrad on the hive workshop. The first level of the campaign is available for download below; you'll need to make sure you have Warcraft III installed to use it.
