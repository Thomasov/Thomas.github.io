Auburn Global Testing Suite

Fall 2015

AU Global is a web application used to give placement or admission exams to students overseas that are seeking to attend Auburn University. It is a C# .NET site that is heavily driven by Knockout.js. The client is free to add a test, or edit any questions on an existing test, at their leisure. The site also provides extensive reporting for the administrators to see which questions are frequently missed, or to see how a specific individual did on a test they took. To render the questions on the test, LaTeX and a heavily modified version of this equation grapher were used. Unfortunately, due the nature of the application, it's not possible to view the testing page or admin functionality without a valid login, so I've uploaded some screenshots to see some high points.

Olympic Weights

Fall 2015

Olympic Weights is a mobile app written to manage the scheduling of weight rooms between different sports teams at Auburn University. It was written using Apache Cordova and also has an accompanying .NET web application for admin functionality and API calls. Also heavily used is a self written MVC framework designed to quickly make multi-paged Cordova applications. Once fully tested, the plan is to open source the framework for others to use and modify as they will. Since the entire application is behind a login wall, I've uploaded some screenshots of the mobile app to get a feel for it. All of the screenshots are from the iOS version of the app, but there is a working Android version as well.

Auburn Office of Accessibility

July 2014

Auburn University's Office of Accessibility site is a C# .NET web application that I wrote early in my career as a web developer. There's not a lot to it, but I'm proud of a lot of the little things in it. It boasts a dynamic system for adding, removing, and editing data driven pages that the client can easily use without ever touching any code. They can also manage many other aspects of the site on their own, including but not limited to the number of links on the home page, images in the slideshow, and news articles on the home page.

Endless Runner

July 2015

This is an endless runner game I built in Unity 5 by following a tutorial by Jesse Freeman on Lynda.com and then expanding on it. All of the implementation was done in C#. The system was built modularly so that the various components, such as the time manager, object pool, and menu manager, could be reused in other future projects. The artwork is almost exclusively not mine (credit to Jesse Freeman), but I did draw the art for the barrel that sometimes appears as an obstacle and the artwork for the pause screen during a forray I had into pixel art.

Plight of the Tauren

Around 2008

A custom Warcraft III campaign based on the conflict between the Tauren and the Centaur in the Barrens. Created in the WCIII World Editor, it has scripted cut scenes and optional quests. It plays very similarly to the already in-game, Blizzard created, campaigns. All art is owned by Blizzard Entertainment, with the exception of the model of the main character, the Gnome Mage, who was created by user Norinrad on the hive workshop.
